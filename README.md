# iwallpaper

修改unsplashReq.js里面的_accessKey为你在unsplash.com注册成为开发者获取到的key

> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
